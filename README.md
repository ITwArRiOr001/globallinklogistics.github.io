Global Link Logistics – Official Website Portfolio

Website Overview
----------------
This repository contains the complete source code and supporting resources for the official Global Link Logistics website. The project represents a professional, responsive static website designed to showcase expertise in international logistics, with a focus on cold-chain exports of fresh produce to Gulf countries and imports from China to India.

The website is built using pure HTML, CSS, and JavaScript, with an emphasis on performance, clarity, and user-centric design. In addition, Python-based automation scripts are included to demonstrate dynamic PDF generation capabilities used for professional export and import checklists.

Core Features
-------------
• Fully responsive layout optimized for mobile, tablet, and desktop devices  
• Multilingual accessibility using Google Translate (English, Hindi, Arabic, Chinese)  
• Lead generation forms integrated using Web3Forms (no backend dependency)  
• Interactive calculators for logistics-related estimations  
• Downloadable professional PDF resources for exporters and importers  
• Structured information architecture with clear navigation and fast load times  
• Trust and credibility elements such as certifications, badges, and process transparency  

Technology Stack
----------------
Frontend:
• HTML5 for semantic structure  
• CSS3 for custom responsive styling  
• Vanilla JavaScript for interactivity and UI behavior  

Automation and Resources:

Hosting:

PDF Automation Overview
-----------------------
This repository includes Python scripts used to generate downloadable logistics checklists in PDF format. These scripts demonstrate how automation can be applied to create standardized, professional documentation for logistics operations.

Included PDFs:
• Export Checklist – Fresh Produce to Gulf Countries  
• Import Checklist – Goods from China to India  

The scripts are provided for demonstration, academic evaluation, and future scalability purposes.

Design and UX Philosophy
------------------------
The design approach prioritizes professionalism, clarity, and trust. The interface avoids heavy frameworks and unnecessary visual clutter, ensuring fast performance and long-term maintainability.

Key principles:
• Clean and mature visual language suitable for logistics and trade professionals  
• Mobile-first and accessibility-aware layout  
• Clear call-to-actions and structured content flow  
• Lightweight implementation with full control over performance  

Deployment Instructions
-----------------------
1. Push all files to the main branch of the repository  
2. Navigate to Repository Settings → Pages  
3. Select source as main branch and root directory  
4. Save settings and wait for deployment  
5. Access the live site via the GitHub Pages URL
